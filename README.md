A clean and simple web-based dictionary application that allows users to search for English word definitions. This project utilizes a public dictionary API to fetch and display word meanings, phonetics, and audio pronunciations.

---

## ‚ú® Features

* **Word Search:** Enter any English word to get its definition.
* **Phonetic Pronunciation:** Displays the phonetic spelling of the word.
* **Audio Pronunciation:** Listen to the correct pronunciation of the word where available.
* **Multiple Meanings:** Shows definitions across different parts of speech (e.g., noun, verb).
* **Usage Examples:** Provides example sentences to give context to the word's meaning.
* **Responsive Design:** The interface is designed to work on both desktop and mobile devices.
* **Not Found Handling:** Gracefully handles cases where a word cannot be found.
