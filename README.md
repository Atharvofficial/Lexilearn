## ğŸ¯ How It Works

1. **Homepage Experience:**
   - Upon loading, Lexilearn fetches 8 random words from the Random Word API
   - Each word card displays a preview with the word, phonetic spelling, part of speech, and a short definition
   - Click any card to view complete details

2. **Search Functionality:**
   - Enter a word in the search bar and press Enter or click Search
   - The app fetches data from the Free Dictionary API
   - Results display with full definitions, audio pronunciations, and examples
   - Scroll automatically returns to the top for easy reading

3. **Word Details:**
   - Complete phonetic transcription
   - Multiple audio pronunciations with different accents
   - All meanings categorized by part of speech
   - Real-world usage examples for context
